# -West-Africa-Security-Incident-Analytics-Dashboard-1974-2020-
This project presents an interactive Power BI dashboard analyzing security incidents across West Africa from 1974 to 2020. It visualizes trends in incidents, fatalities, injuries, attack types, weapon types, and perpetrator groups, offering deep insights into regional security patterns.
